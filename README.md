# Инструкция для личных сообщений
**Всем привет!** Эта инструкция сделана для того, чтобы сделать общение более <ins>содержательным</ins>.

### Как писать не надо?
You: Привет <br/>
*Ждет ответа* <br/>
I: Привет, что хотел? <br/>
You: О, короче <br/>
You: Тут такое дело <br/>
*Прочий спам, который долго идет до самой сути*

### Какой формат приветствуется?
You: Привет, у меня возник вопрос по поводу *Тема*. <br/>
I: Привет, да, давай помогу. <br/>

## Итог?
Я сохраняю время своё и ваше, чтобы я мог быстрее ответить, а вы лучше понять, не дожидаясь следующего моего ответа.
